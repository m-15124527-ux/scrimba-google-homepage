# Google Homepage Clone

This is a simple Google homepage clone built using HTML and CSS as part of the Scrimba Learn HTML & CSS course.

## 🚀 Project Overview

The goal of this project was to recreate the Google homepage layout using basic web development skills. This includes working with HTML structure, CSS styling, and Flexbox for alignment.

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox

## 🎯 What I Learned

- How to structure a webpage using HTML
- How to style elements using CSS
- How to center elements using Flexbox
- How to use margins, padding, and spacing
- How to inspect elements using Chrome DevTools
